# BuildMaster Pro - SaaS Workflow Documentation

This document describes the operational flow of the **Construction Management SaaS Mobile Application**.

## 🏗️ 1. Authentication & Workspace
*   **Login**: Access existing company workspaces.
*   **Company Registration**: Create a new SaaS workspace with company logo and owner details.
*   **Role-based Access**: Interface adapts based on whether the user is an Owner, PM, or Worker.

## 📊 2. Dashboard & Daily Operations
*   **Key Stats**: View total active projects, tasks, and outstanding invoices.
*   **Clock In / Clock Out**: Field-friendly button for staff attendance tracking.
*   **Sync Status**: Real-time indicator for offline/online data availability.

## 📂 3. Project Management (Deep Navigation)
Each project contains a specialized **Material Top Tab** system:
1.  **Overview**: Budget vs. Actual, Team List, Estimates, Invoices, and Purchase Orders.
2.  **Schedule**: Task-by-task breakdown with status chips.
3.  **Photos**: Site gallery with automatic Employee/Timestamp/GPS tagging.
4.  **Drawings**: Versioned access to site plans and PDF blueprints.
5.  **Issues**: High-priority issue logging with assignment tracking.
6.  **Chat**: Modern messaging interface for site-specific coordination.

## 📸 4. Field Capture (Upload)
*   User-friendly interface for site workers to quickly snap photos.
*   Link photos directly to specific projects and tasks.
*   Add field notes on the fly.

## 👤 5. Profile & Cloud
*   Manage user roles and company affiliation.
*   Review subscription plans (Enterprise/Pro/Basic).
*   Account settings and security.
